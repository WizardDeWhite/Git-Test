2. How it works
================================================================================

    global section
    job section[s]

3. Job File Format
================================================================================
https://fio.readthedocs.io/en/latest/fio_doc.html#job-file-format

[] brackets define the job name
each line defines a parameter of the job

Refer example/ directory for inspiration.
