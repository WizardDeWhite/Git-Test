1. Rsync Daemon

1.1 Daemon Config

$ cat /etc/rsyncd.conf
pid file = /var/run/rsyncd.pid
lock file = /var/run/rsync.lock
log file = /var/log/rsync.log
port = 12000

[files]
path = /home/public_rsync
comment = RSYNC FILES
read only = true
timeout = 300

1.2 Start Daemon

$ rsync --daemon

1.3 Strop Daemon

$ kill `cat /var/run/rsyncd.pid`

1.4 Testing Rsync Directories

$ rsync -rdt rsync://127.0.0.1:12000/
files          	RSYNC FILES

$ rsync -rdt rsync://127.0.0.1:12000/files
drwxrwxr-x          4,096 2021/01/05 20:34:03 .
$ touch /home/public_rsync/test
$ rsync -rdt rsync://127.0.0.1:12000/files
drwxrwxr-x          4,096 2021/01/05 20:40:40 .
-rw-rw-r--              0 2021/01/05 20:40:40 test

1.5 Pull the file to local Directory

$ rsync -rdt rsync://127.0.0.1/files ./
